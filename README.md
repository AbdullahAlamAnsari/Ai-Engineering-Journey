Every module below gets its own folder with notes, exercises, and hands-on projects as I work through them.

---

## 📚 Curriculum

<details>
<summary><b>1. Python for AI</b></summary>

- Python basics: history, features, expressions, operators, type inference
- Data structures: lists, tuples, dictionaries, sets, strings
- Conditional statements, `for` loops, `while` loops, `break`/`continue`, nested loops
- Functions: parameters, arguments, scope, return values, documentation
- Exception handling and file handling (try/except/else/finally, custom exceptions, file I/O)
</details>

<details>
<summary><b>2. Mathematics</b></summary>

- **Linear Algebra:** vectors, matrices, NumPy operations, matrix multiplication, determinants, solving linear equations
- **Probability & Statistics:** probability theory, distributions (normal, binomial, Poisson), descriptive statistics, inferential statistics, Central Limit Theorem, hypothesis testing
- **Correlation & Regression:** scatter plots, correlation coefficients, simple linear regression
- **Tools:** pandas, SciPy, Seaborn, Matplotlib for statistical visualization
</details>

<details>
<summary><b>3. Exploratory Data Analysis (EDA)</b></summary>

- Missing value analysis, imputation techniques
- Data consistency checks (fuzzy logic), binning, discretization, outlier detection
- Feature selection (filter, wrapper, embedded methods) and data wrangling
- Statistical inference and hypothesis testing (t-test, chi-square, ANOVA)
- Visualization techniques: histograms, box plots, scatter plots
- Hands-on: full EDA pipeline on a real dataset
</details>

<details>
<summary><b>4. Machine Learning and Deep Learning</b></summary>

- Supervised, unsupervised, and reinforcement learning overview
- ML workflow: data collection → deployment
- Linear regression, logistic regression, Naive Bayes
- SVM, Decision Trees, Random Forests, Bagging & Boosting
- Clustering: K-means, K-medoids, hierarchical, DBSCAN, OPTICS
- Cluster evaluation: silhouette coefficient, Davies-Bouldin index
- Introduction to Deep Learning + Neural Network from scratch
- Hyperparameter tuning strategies
</details>

<details>
<summary><b>5. Multimodal Generative AI — Agents</b></summary>

- Generative AI and multimodal models overview, OpenAI API
- Text-to-speech / speech-to-text, ElevenLabs, HeyGen
- Advanced agent development with **LangChain**
- Building intelligent agents with **CrewAI**
- Multimodal content generation (text, image, speech) with Synthesia + OpenAI
- Final project: multimodal generative application
</details>

<details>
<summary><b>6. No-Code AI Agents</b></summary>

- **n8n:** drag-and-drop workflow automation, API/webhook integration
- **Power Automate Desktop:** business workflow automation, Microsoft 365 integration
- **Azure AI Foundry:** no-code AI model deployment, cognitive services
- **Microsoft Copilot Studio:** custom Copilot workflows, conversational AI for Teams/Office
- Hands-on builds across all four platforms
</details>

<details>
<summary><b>7. Data Warehousing and ETL</b></summary>

- Data scientist vs. data engineer roles
- Data sourcing with Python (PostgreSQL, MongoDB, APIs, web scraping, IoT)
- Databricks for big data processing
- Apache Spark for distributed large-scale data handling
- MLflow for experiment tracking and reproducibility
</details>

<details>
<summary><b>8. Computer Vision</b></summary>

- Convolutional Neural Networks (CNNs): architecture and image processing
- Custom image classification models
- Transfer learning and object detection
- YOLO (You Only Look Once) — hands-on and custom training
- OpenCV: image preprocessing, built-in algorithms
- Capstone: guided project using OpenCV
</details>

<details>
<summary><b>9. Natural Language Processing</b></summary>

- Text normalization, tokenization, stopword removal
- Bag of Words, Count Vectorizer, TF-IDF
- Stemming, lemmatization, N-gram models
- Markov models, language model evaluation, Naive Bayes sentiment classification
- Vector semantics, embeddings (Word2Vec, GloVe)
- Deep learning for NLP and sequence models
- Transformers and LLM architecture
- Training and fine-tuning LLMs
</details>

<details>
<summary><b>10. Building and Deploying AI Systems</b></summary>

- MLOps lifecycle vs. DevOps, automation, reproducibility, scalability
- Git & version control for ML workflows
- Docker & containerization for ML/LLM projects
- Weights & Biases: experiment tracking, model monitoring, dataset logging
- CI/CD with GitHub Actions
- FastAPI: REST APIs for ML/LLM models, model loading, containerized endpoints
- Backend + frontend integration with W&B model fetching
- Deployment on Google Cloud (Cloud Run, App Engine, gcloud CLI)
- Final project: full pipeline orchestration (data → train → API → deploy)
</details>

<details>
<summary><b>11. Professional Development</b></summary>

- Professional email writing and etiquette
- Data science report writing and presentations
- LinkedIn optimization for data science/AI roles
- Resume/CV tailoring
- Freelancing: Upwork setup, pricing, client management
- Kaggle for data science, GitHub portfolio building
- Interview preparation (technical, behavioral, case studies)
- Global market understanding, data commons
- AI product development lifecycle
</details>
